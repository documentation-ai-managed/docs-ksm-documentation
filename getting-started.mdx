## Overview

Set up your ksm documentation space in minutes. Follow these instructions to create an account, build your first space, explore the interface, and upload content. ksm provides an intuitive platform for organizing Markdown and MDX files with powerful preview and collaboration features.

<Callout kind="tip">
  Use your brand color `#3B82F6` in custom themes for a personalized look.
</Callout>

## Create an Account and Log In

Begin by signing up for ksm.

<Steps>
  <Step title="Visit ksm" icon="globe">
    Open your browser and go to `https://ksm.dev`.
  </Step>
  <Step title="Sign Up" icon="user-plus">
    Click **Sign Up** and enter your email, name, and a strong password. Verify your email via the confirmation link.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the login page, enter your credentials, and access the dashboard.
  </Step>
</Steps>

## Create Your First Documentation Space

Organize your project docs in a dedicated space.

<Steps>
  <Step title="New Space" icon="plus">
    From the dashboard, click **New Space**. Name it `My Project Docs` and select a template like "Default MDX".
  </Step>
  <Step title="Configure Settings" icon="settings">
    Set visibility to **Public** or **Private**, add your brand color `#3B82F6`, and save.
  </Step>
  <Step title="Publish" icon="globe">
    Click **Publish** to generate your space URL, such as `https://ksm.dev/my-project-docs`.
  </Step>
</Steps>

## Upload Initial Content

Add your first pages using these methods.

<Tabs>
  <Tab title="Web Editor" icon="edit">
    Create `getting-started.mdx` directly in the editor.

    ```mdx
    ---
    title: My First Page
    description: Welcome to my docs.
    ---

    ## Hello World

    Start writing here.
    ```

    Save and preview instantly.
  </Tab>
  <Tab title="Git Sync" icon="git-pull-request">
    Connect your GitHub repo.

    <CodeGroup tabs="CLI,Web">
    ````bash
    npx ksm-cli init my-docs
    git remote add origin https://github.com/user/my-docs.git
    git push -u origin main
    ````

    ````mdx
    In ksm dashboard: **Settings > Git Sync > Connect GitHub**.
    ````
    </CodeGroup>
  </Tab>
  <Tab title="Drag & Drop" icon="upload">
    Drag Markdown or MDX files into the file tree. ksm auto-generates frontmatter if missing.
  </Tab>
</Tabs>
