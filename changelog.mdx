<Update label="2024-11-05" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added `<Tabs>` and `<CodeGroup>` components for multi-platform examples
- Introduced `<Request>` and `<Response>` for API documentation previews

## Improvements

- Enhanced MDX parsing to support nested code blocks with 4 backticks
- Optimized rendering performance by `>30%` for large documentation sites

## Bug Fixes

- Fixed special character escaping for `{`, `}`, `<`, `>` in prose
- Resolved heading hierarchy validation to prevent H1 usage errors

</Update>

## Integrate Changelog Programmatically

Fetch the latest updates in your application using our API.

<CodeGroup tabs="JavaScript,Python">

```javascript
const response = await fetch('https://api.ksm.dev/changelog');
const updates = await response.json();
updates.forEach(update => console.log(`${update.version}: ${update.summary}`));
```

```python
import requests

response = requests.get('https://api.ksm.dev/changelog')
updates = response.json()
for update in updates:
    print(f"{update['version']}: {update['summary']}")
```

</CodeGroup>
