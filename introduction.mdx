## What is ksm Documentation

ksm Documentation provides a centralized space to organize, manage, and publish your project's documentation. You create structured pages, embed interactive components, and collaborate with your team in real-time. The platform supports MDX for rich content including code examples, diagrams, and custom components.

<Callout kind="info">
  ksm Documentation integrates seamlessly with your Git workflow, so you version-control docs alongside your codebase.
</Callout>

## Brand Customization

Customize your docs to match your brand, starting with the primary color `#3B82F6`.

<CodeGroup tabs="config.json,env">
  ```json
  {
    "theme": {
      "primaryColor": "#3B82F6",
      "logo": "/assets/logo.svg"
    }
  }
  ```
  ```bash
  KSM_PRIMARY_COLOR=#3B82F6
  KSM_LOGO_PATH=/assets/logo.svg
  ```
</CodeGroup>

<Expandable title="Advanced Theming" default-open="false">
  Override CSS variables for deeper customization:
  
  ```css
  :root {
    --ksm-primary: #3B82F6;
    --ksm-secondary: #1D4ED8;
  }
  ```
</Expandable>
